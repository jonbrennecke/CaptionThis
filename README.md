## "Caption This" app

#### Description:

The "Caption This" app for iOS lets you add real-time captions to videos for Instagram Stories.

#### Usage:
Sample commands:
```bash
npm run ios # Run on iOS
npm run android # Run on Android
```

#### Tests:
Sample commands:
```bash
npm run test # Run unit tests with Jest
```
